Tutorin-N-More Project

In this project I was given the task to develop a single page website. I was learning a new CSS framework at the time (Bulma) and was ready to dive in, since this was my first REAL project. This client was in need of an updated logo (something that looked a little more modern without losing its original look) and also an actual website to put it on.

The goal for the website was to have something quick and simple to view as reference to the client's services that they provide (tutorial services - all subjects). We were looking to increase her average class size from 5 to 7 or 8 students.

Result: We improved class size from an average of 5 to 6.
